Este repositório contém um projeto exemplo com **frontend**, **backend** e integração com uma **API**.  
O objetivo é demonstrar como rodar o código localmente e como organizar o fluxo de desenvolvimento.

---

# Tecnologias utilizadas
- **Frontend:** React / Next.js
- **Backend:** Node.js / Express
- **API:** REST / GraphQL
- **Banco de Dados:** PostgreSQL / MongoDB
- **Controle de versão:** GitHub

---

# Estrutura do projeto
/frontend   → Aplicação web (React/Next.js)
/backend    → API e lógica de negócio (Node/Express)


---
